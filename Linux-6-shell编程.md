### 一、作业要求

编写一个 Shell 脚本程序，实现一个简单的 Linux 系统信息与文件管理工具。脚本运行后，应显示菜单，用户可以通过输入编号选择不同功能。

脚本文件命名为：

```
linux_tool.sh
```

运行方式：

```
bash linux_tool.sh
```

或赋予执行权限后运行：

```
chmod+x linux_tool.sh
./linux_tool.sh
```

### 二、具体任务 

一个shell脚本试下如下功能

### 功能 1：显示系统基本信息

输出当前系统的基本信息，包括：

```
当前用户名
当前主机名
当前系统时间
当前工作目录
系统内核版本
系统运行时间
```

可使用命令参考：

```
whoami
hostname
date
pwd
uname-r
uptime
```

---

### 功能 2：查看磁盘与内存使用情况

输出当前系统的磁盘空间和内存使用情况。

要求：

```
显示磁盘使用情况
显示内存使用情况
输出结果格式清晰
```

可使用命令参考：

```
df-h
free-h
```

---

### 功能 3：文件搜索

用户输入一个目录路径和一个文件名关键字，脚本在指定目录下搜索包含该关键字的文件。

示例：

```
请输入搜索目录：/home/student
请输入文件名关键字：test
```

可使用命令参考：

```
find
```

要求：

```
判断目录是否存在
如果目录不存在，给出错误提示
如果找到文件，显示完整路径
如果未找到文件，给出提示信息
```

### 功能4：退出程序

用户选择退出时，脚本应正常结束，并输出提示信息。

### 三、菜单示例

脚本运行后可显示如下菜单：

```
====================================
 Linux 系统信息与文件管理工具

1. 显示系统基本信息
2. 查看磁盘与内存使用情况
3. 文件搜索
4. 退出程序
====================================
请输入功能编号：
```
